Todo カウントダウン - Windows版 使用方法
========================================

■ 必要な環境
- Windows 10/11
- Node.js (https://nodejs.org/ からダウンロード)

■ ファイル説明

1. install_and_run.bat
   - 初回起動用（Node.jsチェック + サーバー起動 + ブラウザ起動）
   - これを実行すれば全て自動で起動します

2. start_server.bat
   - サーバーのみ起動
   - ポート変更も可能

3. open_app.bat
   - ブラウザでアプリを開く
   - サーバーが既に起動している場合に使用

■ 使い方

【初回起動】
1. install_and_run.bat をダブルクリック
2. Node.js がない場合は、インストールを促されます
3. 自動的にサーバーが起動し、ブラウザが開きます

【2回目以降】
- install_and_run.bat をダブルクリック（推奨）
または
- start_server.bat でサーバー起動 → open_app.bat でブラウザを開く

【終了方法】
- サーバーウィンドウで Ctrl+C を押す
- またはウィンドウを閉じる

■ トラブルシューティング

Q: 「ポート3030は既に使用中」と表示される
A: 別のポート番号を入力するか、既存のサーバーを終了してください

Q: ブラウザが開かない
A: http://localhost:3030 を手動でブラウザに入力してください

Q: 「Node.js がインストールされていません」と表示される
A: https://nodejs.org/ から Node.js をダウンロードしてインストールしてください

■ 注意事項
- ファイアウォールの警告が出た場合は「アクセスを許可」してください
- このアプリはローカル環境でのみ動作します
- データはブラウザのローカルストレージに保存されます
